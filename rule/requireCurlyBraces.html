    requireCurlyBraces

    Requires curly braces after statements.

    Types: Array or Boolean

    Values: Array of quoted keywords or true to require curly braces after the following keywords:

    JSHint: curly

    Example

    "requireCurlyBraces": [
    "if",
    "else",
    "for",
    "while",
    "do",
    "try",
    "catch",
    "case",
    "default"
]
    Valid
    if (x) {
    x++;
}
    Invalid
    if (x) x++;
